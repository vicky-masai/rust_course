# Module 08 — Unsafe Rust

`unsafe` doesn't turn off the borrow checker — it lets you do five extra things the compiler can't verify, and makes YOU responsible for the safety proof. This module teaches you to use that power correctly and sparingly.

**Prerequisites:** Modules 01–05.
