# Module 16 — Database Engineering

Real backends live and die by their database usage. This module covers SQLx and connection pooling, then the hard parts: transactions, isolation levels, deadlocks, locking strategies, and how PostgreSQL actually works.

**Prerequisites:** Modules 06–07 and 15.
