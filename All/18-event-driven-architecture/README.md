# Module 18 — Event-Driven Architecture

Systems that communicate through events instead of direct calls scale better and fail more gracefully — if you get the patterns right. This module covers the brokers (Kafka, RabbitMQ, NATS) and the patterns (event sourcing, CQRS, outbox, sagas, idempotency).

**Prerequisites:** Modules 16–17.
