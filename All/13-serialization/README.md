# Module 13 — Serialization

Serialization turns your structs into bytes and back. This module covers Serde (the framework behind almost all Rust serialization) and the formats you'll meet in production: JSON, Bincode, Protobuf, and FlatBuffers.

**Prerequisites:** Modules 03 and 12 (traits and derive macros).
