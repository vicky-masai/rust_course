# Module 01 — Ownership & Memory Model

This is where Rust becomes Rust. Ownership is how Rust manages memory without a garbage collector and without you calling `free()` by hand. Master this module and half of Rust's "weirdness" disappears — the borrow checker stops being an enemy and becomes a code reviewer that never sleeps.

**Prerequisites:** basic Rust syntax (variables, functions, structs).
