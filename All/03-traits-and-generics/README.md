# Module 03 — Traits & Generics

Traits are Rust's answer to interfaces, and generics are how you write code once and use it with many types — with zero runtime cost. This module goes from everyday trait bounds all the way to GATs, auto traits, and specialization.

**Prerequisites:** Modules 01–02.
