# Module 19 — Performance Engineering

"Rust is fast" is only true if your code is. This module teaches you to measure before you optimize: benchmarking with Criterion, profiling, flamegraphs, and the two biggest wins in practice — cache friendliness and fewer allocations.

**Prerequisites:** Modules 01 and 10 (memory layout and collections internals).
