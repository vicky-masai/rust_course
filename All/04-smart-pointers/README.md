# Module 04 — Smart Pointers

Smart pointers are structs that act like pointers but carry extra powers: heap allocation (`Box`), shared ownership (`Rc`, `Arc`), interior mutability (`Cell`, `RefCell`), and thread-safe access (`Mutex`, `RwLock`). Knowing which one to reach for — and why — is a core Rust skill.

**Prerequisites:** Modules 01–03.
