# Module 05 — Concurrency

Rust's big promise: "fearless concurrency." The compiler catches data races before your code ever runs. This module covers threads, channels, shared state, and goes deep into atomics, memory ordering, and lock-free programming.

**Prerequisites:** Modules 01–04 (especially Arc and Mutex).
