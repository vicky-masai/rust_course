# Module 11 — Error Handling

Rust has no exceptions — errors are values you must handle. This module covers `Result` and `Option` deeply, then the ecosystem tools (`thiserror`, `anyhow`, `eyre`) and how to design error types for libraries vs applications.

**Prerequisites:** Modules 01–03.
