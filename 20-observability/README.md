# Module 20 — Observability

You can't fix what you can't see. This module wires up the full observability stack for Rust services: structured logs and spans with `tracing`, metrics with Prometheus, dashboards with Grafana, and request tracing across services with OpenTelemetry.

**Prerequisites:** Modules 07 and 15.
