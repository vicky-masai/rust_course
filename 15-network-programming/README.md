# Module 15 — Network Programming

From raw TCP sockets up to HTTP/3 and gRPC. You'll build servers and clients by hand first, then meet the production tools (Hyper, Tonic) and understand exactly what they do for you.

**Prerequisites:** Modules 06–07 (Async Rust and Tokio).
