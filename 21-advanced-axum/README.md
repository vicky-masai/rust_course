# Module 21 — Advanced Axum

Axum is the go-to web framework for Rust backends. This module goes past "hello world" into production territory: middleware, Tower layers, custom extractors, real authentication and authorization, rate limiting, and long-lived connections.

**Prerequisites:** Modules 07, 15, and 16.
